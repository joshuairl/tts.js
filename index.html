Getting Started

TTSManager is the manager tts.js use to handle its TTS services. It is initialized with several Speaker instances.

// Pass one or more speakers into manager
var ttsmanager = new TTSManager([NativeSpeaker, BaiduSpeaker]);

Each Speaker instance represents each TTS service that TTSManager can use. tts.js provide several built-in TTS Speakers, and you can build your own Speaker class by just implementing a few functions and properties in order to be operated by TTSManager.

Till now, tts.js provide these Speaker.

When you get a TTSManager instance, the only thing you need to do is call the .speak function:

var str = $('.example-speak-input').val();
ttsmanager.speak(str, function() {
  console.log('speaking finish');
});

And tts.js can read sentences one by one, we call them as a 'book'. Use .speakBook function to read a book:

var strs = $('.example-speak-book span').map(function() {
  return $(this).html();
});
ttsmanager.speakBook(strs, function() {
  console.log('speaking finish');
}, {
  onChapterStart: function() {
  },
  onChapterEnded: function() {
  }
});
