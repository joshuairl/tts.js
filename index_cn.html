Getting Started

TTSManager是tts.js用来管理内部的TTS服务的对象。它根据传入的若干个Speaker构造函数来初始化。

// Pass one or more speakers into manager
var ttsmanager = new TTSManager([NativeSpeaker, BaiduSpeaker]);

每个Speaker实例代表了TTSManager能使用的每种TTS服务。tts.js提供了若干种内建的TTSSpeakers，你也可以通过实现几个函数和属性来编写自己的Speaker类，并使它们能TTSManager使用。

目前，tts.js提供了几种Speaker，可以点击Speaker支持列表查看。

当你获得了一个TTSManager实例后，你唯一要做的就是调用它的.speak函数：

var str = $('.example-speak-input').val();
ttsmanager.speak(str, function() {
  console.log('speaking finish');
});

example_speak

并且，tts.js可以一句一句地读句子，我讲这个集合称为一本“书”(Book)。可以使用.speakBook函数来读一本书：

var strs = $('.example-speak-book span').map(function() {
  return $(this).html();
});
ttsmanager.speakBook(strs, function() {
  console.log('speaking finish');
}, {
  onChapterStart: function(source, index) {
    $(".example-speak-book-text span[tts-index=" + index + "]")
      .addClass('speaking');
  },
  onChapterEnded: function() {
    $(".example-speak-book-text span")
      .removeClass('speaking');
  }
});
